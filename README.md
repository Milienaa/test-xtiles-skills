# test-xtiles

A Claude Code plugin that breaks down ideas into actionable tasks and saves them directly to your xTiles workspace.

## Setup

```bash
export XTILES_API_TOKEN=your_token_here
```

## Install

```
/plugin marketplace add Milienaa/test-skills
/plugin install test-xtiles@test-xtiles-marketplace
/reload-plugins
```

## Test

```
/testXTiles:action-plan Launch a new product feature: user onboarding flow
```

Or naturally:

```
Create an action plan for launching user onboarding feature
```
